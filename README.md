# 3R English Study

## 学习笔记记录说明

这个仓库用于记录我的英语学习笔记、阅读心得、词汇积累和练习总结。README 里会作为学习记录的目录与方法说明，方便后续复习和整理。

## 目录结构建议

- `README.md` - 学习笔记总览与使用说明
- `docs/` - 课程资料、讲义、课堂笔记和参考资源
- `notes/` - 按主题或日期分类的学习笔记
- `vocab/` - 词汇表和记忆卡片
- `exercises/` - 英语练习、写作和翻译记录

> 课程资料统一放在 `docs/`，Homework 和任务类内容建议通过 GitHub `issues` 管理。

## 学习笔记记录格式

建议每篇笔记包含以下内容：

1. 学习日期
2. 学习主题
3. 主要内容摘要
4. 新词/短语
5. 练习题或复盘
6. 自我评价与下次计划

示例：

- 日期：2026-04-05
- 主题：英语阅读理解练习
- 摘要：完成了一篇短文阅读，重点掌握了文章主旨和细节信息。
- 新词：context, infer, summarize
- 练习：回答问题 / 改写句子 / 翻译段落
- 计划：下一次复习这些单词，并练习听力理解。

## 学习方法建议

- 每天坚持记录学习内容
- 及时整理新词和例句
- 结合听说读写进行复习
- 定期回顾之前的笔记，做知识巩固

## 如何使用这个仓库

1. 在根目录创建 `docs/`、`notes/`、`vocab/`、`exercises/` 文件夹
2. 用 Markdown 格式记录每日学习笔记
3. 将课程资料、讲义和参考资料保存在 `docs/`
4. 通过 GitHub `issues` 记录 homework、练习安排和问题追踪
5. 通过 Git 管理笔记版本，方便回顾成长轨迹
